## 一、权限配置

### 进入路径
`飞书开放平台` → `应用管理` → `企业自建应用` → 选择你的机器人 → `权限管理`

### 需要开通的权限

| 序号 | 权限名称 | 权限标识 | 用途说明 |
|------|----------|----------|----------|
| 1 | 获取用户 userID | `contact:user.id` | 识别发送者 |
| 2 | 获取用户 OpenID | `contact:user.openid` | 用户身份标识 |
| 3 | 发送消息 | `im:message` | 机器人主动发消息 |
| 4 | 接收消息 | `im:message.receive` | 接收用户消息和文件 |
| 5 | 云空间只读 | `drive:file` | 读取云空间文件 |
| 6 | 云空间读写 | `drive:file:write` | 上传/下载文件 |
| 7 | 云文档只读 | `docx:doc` | 读取云文档 |
| 8 | 云文档读写 | `docx:doc:write` | 编辑云文档 |

### 配置步骤
1. 点击「**添加权限**」
2. 搜索上述权限名称或标识
3. 勾选后点击「**确定**」
4. 点击「**发布**」使权限生效

⚠️ **注意**：部分权限需要管理员审核，发布后等待审核通过

## 二、消息卡片配置（时间回填）

### 进入路径
`功能配置` → `Bot 功能` → `消息卡片`

### 时间选择器组件代码

```json
{
  "type": "date_picker",
  "tag": "date_picker",
  "text": {
    "type": "plain_text",
    "content": "选择日期"
  },
  "initial_date": "",
  "confirm": true,
  "action_id": "date_select"
}
```

### 日期时间选择器代码

```json
{
  "type": "datetime_picker",
  "tag": "datetime_picker",
  "text": {
    "type": "plain_text",
    "content": "选择日期时间"
  },
  "initial_datetime": "",
  "confirm": true,
  "action_id": "datetime_select"
}
```

### 完整卡片示例（带时间回填）

```json
{
  "config": {
    "wide_screen_mode": true
  },
  "header": {
    "template": "blue",
    "title": {
      "type": "plain_text",
      "content": "时间信息收集"
    }
  },
  "elements": [
    {
      "tag": "date_picker",
      "text": {
        "type": "plain_text",
        "content": "选择日期"
      },
      "initial_date": "",
      "action_id": "pick_date"
    },
    {
      "tag": "datetime_picker",
      "text": {
        "type": "plain_text",
        "content": "选择具体时间"
      },
      "initial_datetime": "",
      "action_id": "pick_datetime"
    },
    {
      "tag": "input",
      "text": {
        "type": "plain_text",
        "content": "备注说明"
      },
      "multiline": true,
      "action_id": "input_note"
    }
  ]
}
```

## 三、事件订阅配置

### 进入路径
`功能配置` → `事件订阅`

### 需要订阅的事件

| 事件名称 | 事件类型 | 用途 |
|----------|----------|------|
| 接收消息 | `im.message.receive_v1` | 接收用户发送的消息和文件 |
| 交互回调 | `interactive_menu` | 接收卡片组件交互（如时间选择） |
| 消息已读 | `im.message.read` | 可选，追踪消息阅读状态 |

### 配置步骤
1. 点击「**添加事件**」
2. 搜索事件类型
3. 勾选后设置回调地址（如有）
4. 点击「**保存**」

## 四、文件接收配置

### 进入路径
`功能配置` → `机器人能力`

### 开启选项

- ✅ **接收文件**：允许用户发送文件给机器人
- ✅ **文件预览**：在聊天中预览文件
- ✅ **文件下载**：支持下载到本地/手机端

### 文件大小限制

| 文件类型 | 最大大小 |
|----------|----------|
| 图片 | 10 MB |
| 文档 | 100 MB |
| 视频 | 500 MB |
| 其他 | 100 MB |

## 五、回调处理示例

### 时间选择器回调数据处理

```python
# 当用户选择时间后，飞书会发送回调
{
    "type": "interactive_menu",
    "action": {
        "value": {
            "date": "2026-03-10",
            "datetime": "2026-03-10 14:30"
        },
        "action_id": "pick_date"
    },
    "user": {
        "open_id": "ou_xxxxx"
    }
}
```

### 文件接收回调数据处理

```python
# 当用户发送文件后
{
    "type": "im.message.receive_v1",
    "message": {
        "message_id": "msg_xxxxx",
        "type": "file",
        "content": {
            "file_key": "file_xxxxx",
            "file_name": "测试文档.xlsx",
            "file_size": 102400
        }
    },
    "sender": {
        "open_id": "ou_xxxxx"
    }
}
```

## 七、常见问题

### Q1: 权限添加后不生效？
**A**: 需要点击「发布」按钮，部分权限需要管理员审核

### Q2: 收不到文件回调？
**A**: 检查是否开通了 `im:message.receive` 权限

### Q3: 时间选择器不显示？
**A**: 检查卡片 JSON 格式是否正确，`tag` 字段必须是 `date_picker` 或 `datetime_picker`

### Q4: 文件无法下载到手机？
**A**: 确保开通了 `drive:file:upload` 权限，且文件大小在限制范围内
